# SimplyEDH
Rebuild of SimplyEDH workshop mod for Tabletop Sim with replaced assets for users where Imgur is not available

## Missing assets
These assets are not able to be found at the time of writing, but YOU might be able to help!!

If you used the original SimplyEDH table on your computer, you may still have these files stored locally. To find out, go to one of the following folders and look for files named similarly to the below.

NOTE, files are stored with all special characters removed, so "`http://cloud-3.steamusercontent.com/ugc/861734447427622808/DB9EEA3B7977B0D4912A0C4C3A76E36FC3A6062E/`" would become "`httpcloud3steamusercontentcomugc861734447427622808DB9EEA3B7977B0D4912A0C4C3A76E36FC3A6062E`"

If you find the files we're looking for, please raise a PR, or email software@brusher.dog

### Folder locations
Windows: C:\Users\\[Username]\Documents\My Games\Tabletop Simulator\Mods\

Mac: ~/Library/Tabletop Simulator/Mods/

Linux (native): ~/.local/share/Tabletop Simulator/Mods/

Linux (proton): ~/.local/share/Steam/steamapps/compatdata/286160/pfx/drive_c/users/steamuser/Documents/My Games/Tabletop Simulator/Mods/
